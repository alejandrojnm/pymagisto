Pymagisto
==============================

A python wrapper for magisto api


Installation
------------
    pip3 install pymagisto
    git clone https://github.com/alejandrojnm/pymagisto.git
    python setup.py install

Usage
-----
    
    from pymagisto import Pymagisto
    data = Pymagisto(API_KEY, API_SECRET)
