Pymagisto

python wrapper for magisto api

Usage
-----
>>> from pymagisto import Pymagisto
>>> data = Pymagisto(API_KEY, API_SECRET)


Installation
------------
pip3 install pymagisto


Requirements
^^^^^^^^^^^^
requests

Compatibility
-------------
python 3.7
